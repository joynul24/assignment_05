What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML documentbr
access, modify, and update the document's content, structure, and styles.

How do you select an element from the DOM?

You can select an element from the DOM using JavaScript with various methods provided by the document object. The choice of
method depends on the type of selector you want to use, such as an ID, class, tag, or a more complex CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where you add an event listener to a parent element rather than individual child elements. The parent element listens for events triggered by its child elements by taking advantage of event propagation, specifically the bubbling phase.

How do you manipulate an elelment's attributes and styles using the DOM?

You can manipulate an element's attributes and styles in the DOM using JavaScript. Here's how you can do it:

1. Manipulating Attributes Common Methods:

getAttribute(): Retrieves the value of a specified attribute.

setAttribute(): Sets or updates the value of a specified attribute.

removeAttribute(): Removes a specified attribute from an element.

Direct property access: Attributes like id, src, or href can often be manipulated directly as properties of the DOM element.
